The RedCrossTweetsCollector is a zipped folder that contains all R code files used to get the red cross tweets. To be run using Rstudio.

TweetCrawler contains all java files that were used to get the civil defense tweets. 

BuckWalterConverter contains the java files to transform the Arabic text to english characters that will be used for future research in getting the location of accidents (tweets) and use it in spatiotemporal analysis to predict time and place of accidents.
